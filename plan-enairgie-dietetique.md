- Base unit : `8px` (multiples : 4, 8, 12, 16, 24, 32, 48, 64, 80, 96)
- CSS Grid 12 colonnes, max-width `1200px`, gutter `24px`
- Breakpoints : `480px` / `768px` / `1024px` / `1440px`

### 🎭 Animations Framer Motion — Stratégie "Zen"

| Animation | Config | Usage |
|---|---|---|
| Fade In Up | `duration: 0.8, ease: "easeOut"` | Sections au scroll |
| Stagger Children | `staggerChildren: 0.15` | Cartes, FAQ |
| Scale Subtle | `scale: [0.97, 1], duration: 0.6` | Boutons hover |
| Blob Morphing | `duration: 8, repeat: ∞` | Formes de fond |
| Float | `y: [-5, 5], duration: 4, mirror` | Éléments décoratifs |
| Card Hover Lift | `y: -4, duration: 0.3` | Cartes interactives |

> 🧘 Principe : animations lentes (> 0.5s), easings organiques, `prefers-reduced-motion` respecté.

## AXE 4 : Stratégie de transition visuelle Diet ↔ Énergétique

### Pattern : **Scroll-Based Gradient Morphing + Split-Screen**

#### 🎨 Zones thématiques au scroll

```
 0%  ┌─────────────┐  Zone Neutre    → #FAF8F5 → #F5E6A3
20%  │   Hero       │
     │ Storytelling │
     ├─────────────┤  Zone Diététique → #F5E6A3 → #E8F0E2
40%  │ DualServices │
     │   (gauche)   │
     ├─────────────┤  Transition      → #E8F0E2 → #F2E4E8
60%  │ DualServices │
     │   (droite)   │
     ├─────────────┤  Zone Énergétique → #F2E4E8 → #E8DCF0
80%  │ Ateliers     │
     │ Testimonials │
     ├─────────────┤  Retour Neutre   → #E8DCF0 → #FAF8F5
100% │ Tarifs/FAQ   │
     │ Contact      │
     └─────────────┘
```

#### 🔧 Mécanisme technique

1. **`GradientOverlay`** — Couche `position: fixed; z-index: -1` avec `background` interpolé dynamiquement via `useScrollProgress()` + `interpolateColor()`

2. **Blobs SVG** — Formes différentes par univers :
    - Diet : feuilles, vert sauge 10% opacité, morphing 8s
    - Energy : auras rondes, rose/lavande 10% opacité, morphing 10s
    - Apparition/disparition via `useIntersection` (fade 1s)

3. **DualServices Split-Screen** :
    - `DietetiquePanel` : `initial: { x: -60, opacity: 0 }` → `animate: { x: 0, opacity: 1 }` (0.8s)
    - `EnergetiquePanel` : `initial: { x: 60, opacity: 0 }` → `animate: { x: 0, opacity: 1 }` (0.8s, delay 0.2s)
    - Divider SVG ondulé vertical entre les deux (dégradé vert → rose)

4. **Navbar adaptative** — L'indicateur actif change de couleur :
    - Zone Diet → vert sauge
    - Zone Energy → rose poudré
    - Zone Neutre → or chaud
    - Transition CSS : `transition: border-color 0.5s ease`

5. **WaveDivider** — SVG ondulé entre sections, couleur = section suivante, léger parallaxe au scroll

6. **Cohérence** :
    - Mêmes typos partout (Playfair + Lato)
    - Composants UI avec prop `universe: 'diet' | 'energy' | 'neutral'`
    - Or chaud (`#C9A96E`) = fil conducteur commun
    - `prefers-reduced-motion` désactive toutes les animations
